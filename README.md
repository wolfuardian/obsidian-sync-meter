Show real-time [Obsidian Sync](https://obsidian.md/sync) progress percentage in the status bar.

## Features

- Displays sync progress as a percentage with a mini progress bar in the status bar
- Three visual states:
  - **Syncing** — accent-colored icon + animated progress bar + percentage
  - **Synced** — dimmed icon showing 100%
  - **Sync unavailable** — greyed out when Obsidian Sync is not enabled
- Polls sync status every 2 seconds with minimal overhead
- Works with both desktop and mobile

## Usage

Install and enable the plugin. The sync percentage will appear automatically in the status bar — no configuration needed.
